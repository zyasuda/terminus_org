- 検証は複数シーン・複数プレイスルーで実施。合成プロンプトでの秘密漏洩検証(1/6で漏洩を1件確認)、実プレイでの灯の番人との戦闘死亡事例(初のゲームオーバー)、口調ブレ(ガレスが女性的な語尾で発言)、戦闘一言の連続重複、を発見
- **見つかった不具合はその都度修正済み**:
  - 戦闘中の同行者一言(`battleMutters`)が直前と同じ文を連投する問題 → 直前選出文を候補から除外するガードを追加([src/engine/index.js](../src/engine/index.js))
  - シーン切り替え時に前シーンの同行者/NPC吹き出しが残る問題 → `advanceScene()`で`companionBubbles`/`npcBubble`をクリア([src/engine/index.js](../src/engine/index.js))

- **灯の番人との戦闘死亡事例は「バグ」ではなく仕様通りの挙動と判断**: 「攻撃する」という明示的な宣言をシステムがそのまま実戦闘として処理した結果であり、状態破壊やスキーマ違反はない。ただし`灯の番人`は本来「対話・回避」が想定ルートの敵(`atk:2`, `defenseDc:13`と、通常戦闘の`錆喰い`より高難度)であり、モデルによって同じプレイヤー宣言の解釈(戦闘化するか否か)が変わりうる点は今後の考慮事項として記録
